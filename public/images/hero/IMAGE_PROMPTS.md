## 디자인 가이드라인

### 공통 스타일
1. **색상**: 각 컨텐츠별 지정된 그라데이션 색상 사용
2. **배경**: 어두운 (#0a0a0f ~ #1a1a2f) 계열 선호
3. **효과**: 글로우, 블러, 빛 효과 적극 활용
4. **분위기**: 신비롭고 프리미엄한 앱 느낌
5. **구도**: 중앙 집중형 또는 대각선 구도

### 금지 사항
- 텍스트나 글자 포함 금지
- 과도하게 복잡한 구성
- 현실적인 인물 사진
- 저품질/픽셀화된 이미지
- 다른 브랜드 로고나 상표

### 권장 사항
- 추상적이고 상징적인 이미지
- 동양적 모티브의 현대적 해석
- 고해상도 (최소 1600x1000px 제작 후 리사이즈)
- 어두운 UI에서 잘 보이는 밝은 포인트 요소

## 이미지 적용 방법

코드에서 이미지를 활성화하려면 `HeroPage.tsx` 파일에서 주석을 해제하세요:

```tsx
// 현재 주석 처리된 부분:
{/*
<Image
  src={card.imagePath}
  alt={card.imageAlt}
  fill
  className="object-cover group-hover:scale-105 transition-transform duration-700"
/>
*/}

// 아래와 같이 변경:
<Image
  src={card.imagePath}
  alt={card.imageAlt}
  fill
  className="object-cover group-hover:scale-105 transition-transform duration-700"
/>
```

이미지 파일을 `/public/images/hero/` 폴더에 위에서 지정한 파일명으로 저장하면 자동으로 적용됩니다.
